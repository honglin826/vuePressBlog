## 初始化项目

1. npm init

2. 安装webpack babel

3. 创建webpack.config.js等文件

4. tsc --init 创建tsconfig.json ts配置文件

5. npm run dev 开启本地开发环境

6. npm run build 生成生产环境js文件

## 知识点

* new Map()

* new Set()

* Reflect.set(target, propertyKey, value[, receiver])

允许你在对象上设置属性
